Microsoft IXN Collaboration 10/2021 - 03-2022 This project was completed in collaboration with Microsoft, as part of the UCL IXN programme. The task was to automate workplace processes. More specifically, this project aimed to facilitate the matching of Computer Science Students in their final year with Academics in order to collaborate on a final year project. Along with my two peers, we created a Portal and Model-Driven app to solve this issue. This includes streamlining the unsupervised matching process, along with implementing an algorithm to automatically create these matches based on common interests to be used on remaining students and academics after the unsupervised matching phase has ended.

Linked is the website we used as a portfolio for this project, where we explain more in-depth how we achieved this. Unfortunately, due to GDPR regulations you will not be able to try out the Portal or Model-Driven app for yourself, however I created a short video which shows and explains the project in action which can be found on the homepage of the portfolio.
