Wordle Solver 04/02/2022 A program to solve the game Wordle. Made as a small side project to practice Information Theory, this program uses the Selenium library to navigate to the Wordle website (using a Chrome driver) and will proceed to solve the day's Wordle. *Disclaimer* - the logic of this solver works on an assumption about the way repeated letters are represented on the Wordle website, and assumption which turned out to differ from the way the website actually implements the game. As a result, this program may fail to solve the word if it contains any repeated letters.
Spiral Classification Neural Network 19/10/2021 My first neural network, which classifies points in a spiral. Heavily supervised by the book 'Neural Networks from Scratch in Python' by Harrison Kinsley. An introduction to concepts within Machine Learning such as efficient matrix manipulation, activation functions, different network depths and sizes, and backpropagation using optimisation techniques like momentum, cache, and learning rate decay.
Tetris Autoplayer 20/11/2020 A program which will play tetris. This was a university project - it was a competition to create the highest scoring tetris AI in the year (this project scored in the top 1.3% of my course). To find the optimal weights with which the algorithm evaluates game states, I used a Genetic Algorithm approach; training of the algorithm to find the optimal weights involved running a process mimicking natural selection. Different mutations of the automated player were pitted against each other for around five hours - the weaker players were removed from the competing population whilst the winners remained and were combined to create a stronger new population for the next round. The tetris game itself was pre-made by my professor - I only implemented 'player.py'.
Minesweeper 24/06/2020 A simple minesweeper game using the tkinter library. Created simply to practice my Tkinter skills.
Work Hours Logger 16/05/2020 A simple application to track shifts worked and paychecks earned, to prevent being underpaid. Made by request for a friend, as some practice with the tkinter library.
Chess Engine 15/04/2020 An engine to play and evaluate chess positions, as well as a tkinter interface through which to interact with the program. Uses techniques including minimax, alpha-beta pruning, and quiescence search. Created as A-level Coursework. During testing of difficulty 3 playing 3 minute blitz chess, the AI reached a peak of ~1600 rating on Chess.com.
Noughts and Crosses AI 18/02/2020 A noughts and crosses game in which you play against an impossible-to-beat AI. Created as an introduction to game theory and the minimax algorithm (with alpha-beta pruning) to prepare for making a chess engine.
Sudoku Solver 26/11/2019 An early project in which a file containing a sudoku is read and then (hopefully) solved. Created as an introduction to object-oriented programming.
