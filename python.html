Chess Engine 15/03/2020 An engine to play and evaluate chess positions, as well as a tkinter interface through which to interact with the program. Uses techniques including minimax, alpha-beta pruning, and quiescence search. Created as A-level Coursework. During testing of difficulty 3 playing 3 minute blitz chess, the AI reached a peak of ~1600 rating on Chess.com.
Tetris Autoplayer 20/11/2020 A program which will play tetris. This was a university project - it was a competition to create the highest scoring tetris AI in the year. To find the optimal weights with which the algorithm evaluates game states, I used a Genetic Algorithm approach; training of the algorithm to find the optimal weights involved running a process mimicking natural selection. Different mutations of the automated player were pitted against each other for around five hours - the weaker players were removed from the competing population whilst the winners remained and were combined to create a stronger new population for the next round. The tetris game itself was pre-made by my professor - I only implemented 'player.py'.
